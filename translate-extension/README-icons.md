# 图标说明

Chrome 扩展需要以下尺寸的 PNG 图标：
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

## 获取图标

1. 访问 [Flaticon](https://www.flaticon.com/) 搜索 "translate"
2. 下载 PNG 格式，确保选择 16x16, 48x48, 128x128 三个尺寸
3. 将下载的图标重命名为 icon16.png, icon48.png, icon128.png
4. 放入本目录

或使用在线工具将 icons/translate-icon.svg 转换为 PNG：
- [CloudConvert](https://cloudconvert.com/svg-to-png)
- [OnlineConvert](https://image.online-convert.com/svg-to-png)
