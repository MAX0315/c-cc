## 一、股票基础概念

### 什么是股票？

股票是股份有限公司发行的所有权凭证，购买股票即成为公司的股东，享有公司盈利分红和重大决策投票权。

**核心要点：**
- 股票代表公司净资产的所有权份额
- 股价短期由市场供求决定，长期由公司基本面决定
- 中国A股市场主要有上海证券交易所和深圳证券交易所

## 二、股票涨跌原理

### 2.1 基本供需关系

| 情况 | 结果 |
|------|------|
| 买盘 > 卖盘 | 股价上涨 ↑ |
| 买盘 < 卖盘 | 股价下跌 ↓ |
| 买卖均衡 | 股价横盘 → |

### 2.2 影响股价的核心因素

**基本面因素：**
- 📊 公司业绩：净利润增长 → 股价上涨
- 💰 盈利能力：毛利率、ROE等指标优秀 → 资金青睐
- 📈 行业前景：朝阳行业更受市场关注

**市场面因素：**
- 🌍 宏观经济：GDP增长、货币政策宽松 → 利好股市
- 🏛️ 政策影响：产业扶持政策 → 相关板块上涨
- 💭 市场情绪：投资者信心影响短期走势
- 💵 资金面：北向资金、杠杆资金流向

## 三、K线图基础知识

### 3.1 K线的构成

> K线起源于日本米市，又称"蜡烛图"，由四个价格组成

```
     ┌──┐
     │  │ ← 上影线（最高价）
 ┌───┤  ├───┐
 │   │  │   │ ← 实体（开盘-收盘）
 └───┤  ├───┘
     │  │ ← 下影线（最低价）
     └──┘
```

**四个价格：**
- 开盘价：集合竞价最终成交价
- 收盘价：交易日最后一个成交价
- 最高价：当日最高成交价
- 最低价：当日最低成交价

### 3.2 阳线与阴线

| 类型 | 颜色 | 条件 | 含义 |
|------|------|------|------|
| 阳线 | 红色/空心 | 收盘价 > 开盘价 | 上涨 |
| 阴线 | 绿色/实心 | 收盘价 < 开盘价 | 下跌 |

### 3.3 经典K线形态（实战重点）

**反转形态：**
- 🔨 **锤子线**：下影线是实体的2-3倍，出现在下跌底部 → 买入信号
- ⚡ **上吊线**：锤子线的倒置，出现在上涨顶部 → 警惕回落
- ⭐ **吞没形态**：阳包阴（买入）、阴包阳（卖出）
- 🔄 **十字星**：开盘=收盘，多空博弈平衡 → 观望信号

**持续形态：**
- ➡️ **纺锤线**：实体小、上下影线长 → 盘整信号
- 📦 **三兵形态**：红三兵（强势）、黑三兵（弱势）

## 四、常用技术指标（重点）

### 4.1 均线系统（MA）—— 趋势之王

**什么是均线？**
将一定周期内的收盘价加权平均，连接成曲线

| 周期 | 名称 | 用途 |
|------|------|------|
| MA5 | 一周均线 | 超短线操作参考 |
| MA10 | 两周均线 | 短线趋势判断 |
| MA20 | 一个月 | 中期趋势分水岭 |
| MA60 | 一季度 | 长期趋势确认 |
| MA120 | 半年线 | 牛熊分界线 |

**均线实战用法：**
- ✅ **多头排列**：短周期在上、长周期在下 → 强势上涨
- ❌ **空头排列**：短周期在下、长周期在上 → 强势下跌
- 🔀 **均线金叉**：短周期上穿长周期 → 买入信号
- 🔀 **均线死叉**：短周期下穿长周期 → 卖出信号
- 🔗 **均线粘合**：多周期汇聚 → 行情即将爆发

### 4.2 MACD指标 —— 趋势研判

**MACD构成：**
```
MACD = DIF线 - DEA线
DIF（差离值）= 12日EMA - 26日EMA
DEA（信号线）= DIF的9日加权移动平均
```

**MACD实战用法：**

| 信号 | 形态 | 含义 | 操作 |
|------|------|------|------|
| 金叉 | DIF上穿DEA | 多方占优 | 🟢 买入 |
| 死叉 | DIF下穿DEA | 空方占优 | 🔴 卖出 |
| 零轴上方 | DIF>DEA | 牛市格局 | 逢低买入 |
| 零轴下方 | DIF<DEA | 熊市格局 | 逢高卖出 |

**高级用法 —— 背离：**
- ⬆️ **顶背离**：股价创新高，但MACD没创新高 → ⚠️ 警惕回落！
- ⬇️ **底背离**：股价创新低，但MACD没创新低 → 🔔 关注反弹！

### 4.3 KDJ指标 —— 超买超卖

**KDJ构成：**
- K值：快速随机指标（一般9日）
- D值：K值的平滑移动平均
- J值：3K - 2D，波动最敏感

**实战判断：**
| 区域 | KDJ值 | 含义 | 操作建议 |
|------|-------|------|----------|
| 超买区 | >80 | 股价过热 | 🔴 考虑卖出 |
| 超卖区 | <20 | 股价过冷 | 🟢 关注买入 |
| 中性区 | 20-80 | 正常波动 | 观望 |

**金叉死叉：**
- 🟢 KDJ金叉（K上穿D）+ 在20附近 → 较强的买入信号
- 🔴 KDJ死叉（K下穿D）+ 在80附近 → 较强的卖出信号

### 4.4 布林带（BBAND）—— 波动通道

**布林带构成：**
```
上轨 = MA20 + 2倍标准差（压力位）
中轨 = MA20（趋势线）
下轨 = MA20 - 2倍标准差（支撑位）
```

**实战用法：**

| 形态 | 含义 | 操作 |
|------|------|------|
| 股价触及上轨 | 接近压力 | 🔴 谨慎/卖出 |
| 股价触及下轨 | 接近支撑 | 🟢 关注买入 |
| 布林带收口 | 行情蓄势 | ⏳ 等待突破 |
| 布林带开口 | 行情加速 | 📈 顺势而为 |

### 4.5 VOL成交量 —— 量先于价

> "量为价先"是技术分析的核心原则

**量价配合八律：**

| 情况 | 信号 | 操作 |
|------|------|------|
| 放量上涨 | 资金涌入 | 🟢 持股待涨 |
| 缩量上涨 | 动能不足 | ⚠️ 警惕回调 |
| 放量下跌 | 恐慌抛售 | ⚠️ 谨慎观望 |
| 缩量下跌 | 抛压减轻 | 🔔 底部信号 |
| 放量突破 | 真突破 | 🟢 顺势买入 |
| 缩量突破 | 假突破 | 🔴 警惕陷阱 |
| 地量见地价 | 极度缩量 | 🔔 底部区域 |
| 天量见天价 | 极度放量 | 🔴 警惕顶部 |

**换手率参考：**
- 1-3%：交易正常
- 3-5%：较为活跃
- 5-10%：高度活跃
- >10%：异常活跃，警惕风险

## 五、基本面分析基础

### 5.1 核心财务指标

**估值类：**
| 指标 | 计算公式 | 解读 |
|------|----------|------|
| PE（市盈率） | 股价/每股收益 | 越低越便宜，一般<15为合理 |
| PB（市净率） | 股价/每股净资产 | 越低越有价值，<1为破净 |
| PS（市销率） | 股价/每股销售额 | 适合亏损公司，<3为合理 |

**盈利类：**
| 指标 | 解读 |
|------|------|
| 净利润增长率 | >20%为高增长，<0需警惕 |
| 毛利率 | >30%说明竞争力强 |
| ROE（净资产收益率） | >15%为优秀 |

**财务排雷：**
- ⚠️ 应收账款异常增长
- ⚠️ 存货周转天数大幅增加
- ⚠️ 经营活动现金流持续为负
- ⚠️ 资产负债率超过70%

### 5.2 估值方法

**相对估值法：**
- 与行业平均PE/PB对比
- 与历史平均估值对比

**绝对估值法（DCF）：**
- 预测未来现金流
- 折现到现值
- 适合稳健投资者

**PEG估值法：**
```
PEG = PE / 净利润增长率
PEG < 1：相对低估
PEG = 1：合理估值
PEG > 1：相对高估
```

## 六、趋势分析与支撑压力

### 6.1 三种趋势

| 趋势 | 特征 | 操作策略 |
|------|------|----------|
| 上升趋势 | 高点不断抬高，低点不断抬高 | 逢低买入，顺势而为 |
| 下降趋势 | 高点不断降低，低点不断降低 | 逢高卖出，轻仓观望 |
| 横盘震荡 | 高低点基本持平 | 高抛低吸，快进快出 |

### 6.2 支撑与压力

**重要概念：**
- 🟢 **支撑位**：股价下跌时遇到买盘支撑的区域
- 🔴 **压力位**：股价上涨时遇到卖盘压力的区域
- 🔄 **角色转换**：支撑跌破变压力，压力突破变支撑

**识别支撑压力的方法：**
1. 前期高低点
2. 均线位置（MA20、MA60）
3. 成交密集区
4. 整数关口（3000点、3500点等）

## 七、风险控制（关键！）

> 🎯 风险控制是长期生存的核心！

### 7.1 止损原则

| 方法 | 说明 | 适用 |
|------|------|------|
| 固定止损 | 亏损X%无条件卖出 | 短线操作 |
| 移动止损 | 随价格上涨提高止损线 | 波段操作 |
| 逻辑止损 | 买入逻辑消失即卖出 | 趋势交易 |

**建议：**
- 短线止损：7-10%
- 中线止损：10-15%
- 长线止损：20%

### 7.2 仓位管理

| 仓位 | 操作 | 适用场景 |
|------|------|----------|
| 满仓 | 100%资金买入 | 确定性极高 |
| 重仓 | 60-80% | 明显趋势 |
| 半仓 | 50% | 震荡行情 |
| 轻仓 | 20-30% | 谨慎观望 |
| 空仓 | 0% | 行情不明 |

**仓位口诀：**
> "不该满仓时不满仓，不该空仓时不满仓"

### 7.3 交易纪律

✅ **应该做的：**
- 制定交易计划，按计划执行
- 顺势而为，不要逆势操作
- 分散投资，不要单押一只
- 保留现金，永不满仓
- 记录交易，总结复盘

❌ **不应该做的：**
- 不要追涨杀跌
- 不要频繁交易
- 不要重仓单一股票
- 不要忽视止损
- 不要情绪化交易
