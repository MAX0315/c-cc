# 股票的基本涨跌知识

## 一、股票基础概念

股票是股份有限公司发行的所有权凭证，代表持有者对公司净资产的所有权。股票市场是投资者买卖股票的平台，股票价格由市场供求关系决定。

---

## 二、股票涨跌原理

### 2.1 基本供需关系

- 买盘 > 卖盘时，股价上涨（供不应求）
- 买盘 < 卖盘时，股价下跌（供过于求）
- 买卖盘平衡时，股价基本稳定

### 2.2 影响股价的因素

- 公司业绩：盈利增长推动股价上涨
- 宏观经济：经济增长利好股市
- 政策影响：产业政策、货币政策的松紧
- 市场情绪：投资者心理和预期
- 资金面：流动性充裕推动上涨

## 三、K线图基础知识

### 3.1 K线构成

> K线由开盘价、收盘价、最高价、最低价四个价格组成

- 阳线（红色）：收盘价 > 开盘价，表示上涨
- 阴线（绿色）：收盘价 < 开盘价，表示下跌
- 上影线：最高价到收盘价/开盘价的距离
- 下影线：开盘价/收盘价到最低价的距离

### 3.2 常见K线形态

- 锤子线：底部反转信号
- 上吊线：顶部反转信号
- 十字星：多空平衡信号
- 光头光脚：强烈趋势信号

## 四、常用技术指标

### 4.1 均线（MA）

- 5日均线（MA5）：短期趋势参考，敏感度高
- 10日均线（MA10）：短线交易参考
- 20日均线（MA20）：中期趋势参考
- 60日均线（MA60）：长期趋势参考
- 均线金叉：短期均线上穿长期均线，买入信号
- 均线死叉：短期均线下穿长期均线，卖出信号

### 4.2 MACD指标

- DIF线（差离值）：短期与长期指数移动平均线的差值
- DEA线（信号线）：DIF线的移动平均
- MACD柱：DIF与DEA的差值×2
- **金叉**：DIF上穿DEA，柱由负转正 → 买入信号
- **死叉**：DIF下穿DEA，柱由正转负 → 卖出信号
- **顶背离**：股价创新高但MACD没创新高 → 警惕下跌
- **底背离**：股价创新低但MACD没创新低 → 关注反弹

### 4.3 KDJ指标

- K值：当前股价在近期价格区间中的位置（0-100）
- D值：K值的移动平均（稳定性更高）
- J值：3×K - 2×D（波动最敏感）
- **超买区（>80）**：注意回调风险
- **超卖区（<20）**：注意反弹机会
- KDJ金叉：D线上穿K线 → 买入信号
- KDJ死叉：D线下穿K线 → 卖出信号

### 4.4 布林带（BBAND）

- 上轨：MA20 + 2×标准差（压力位）
- 中轨：MA20（趋势参考）
- 下轨：MA20 - 2×标准差（支撑位）
- 股价触及上轨：注意回落风险
- 股价触及下轨：关注反弹机会
- 布林带收口：行情即将突破
- 布林带开口：行情加速中

## 五、量价关系基础

> "量为价先"，成交量是价格变化的先行指标

### 5.1 量价配合

- **放量上涨**：资金积极入场，看多信号（量增价涨）
- **缩量上涨**：上涨动能减弱，谨慎信号（量缩价涨）
- **放量下跌**：恐慌抛售，但可能加速见底（量增价跌）
- **缩量下跌**：卖压减轻，可能见底（量缩价跌）
- **地量低价**：成交量极度萎缩，往往是底部信号

### 5.2 换手率

- 换手率 = 成交股数/流通股数×100%
- 换手率高：股票交易活跃
- 换手率过低：流动性不足

## 六、基本面分析基础

### 6.1 财务指标

- **市盈率（PE）**：股价/每股收益，越低估值优势越大
- **市净率（PB）**：股价/每股净资产，破净可能价值重估
- **净利润增长率**：企业成长性核心指标
- **毛利率**：企业竞争力体现，毛利率高说明护城河宽
- **ROE（净资产收益率）**：股东权益回报率，越高越优秀
- **资产负债率**：负债/总资产，警惕高负债企业

### 6.2 估值方法

- **相对估值法**：PE、PB、PS等比值与行业平均对比
- **绝对估值法**：DCF（现金流折现）模型
- **PEG估值**：PE/净利润增长率，<1为相对低估

## 七、趋势分析基础

### 7.1 趋势类型

- **上升趋势**：高点不断抬高，低点不断抬高
- **下降趋势**：高点不断降低，低点不断降低
- **横盘震荡**：高低点基本持平

### 7.2 支撑与压力

- **支撑位**：股价下跌时遇到的买盘支撑
- **压力位**：股价上涨时遇到的卖盘压力
- **突破压力位**：需放量确认，否则可能是假突破
- **跌破支撑位**：需警惕加速下跌风险

---

## 八、风险控制基础

- **止损原则**：亏损达到7-10%时果断止损
- **仓位管理**：单只股票不超过总仓位的20%
- **分散投资**：不把鸡蛋放在同一个篮子里
- **右侧交易**：趋势确认后再入场，减少逆势操作
- **保持理性**：避免情绪化交易决策
- **永不满仓**：保留现金应对突发情况
- **不追涨杀跌**：追高风险大，等回调买入

## 九、常见术语速查

| 术语 | 解释 |
|------|------|
| 牛市 | 股价整体上涨的趋势 |
| 熊市 | 股价整体下跌的趋势 |
| 震荡市 | 股价在一定区间内波动 |
| 突破 | 股价向上穿越重要压力位 |
| 回踩 | 股价上涨后回落测试支撑位 |
| 筑底 | 股价在低位逐步企稳的过程 |
| 解套 | 股价回升到成本价以上 |
| 割肉 | 亏损卖出股票 |
| 追高 | 在高位买入股票 |
| 抄底 | 在低位买入股票 |
| 满仓 | 账户资金全部买入股票 |
| 空仓 | 账户没有股票全是现金 |
| 建仓 | 开始买入股票 |
| 加仓 | 买入更多同一只股票 |
| 减仓 | 卖出一部分股票 |
| 清仓 | 卖出全部股票 |
