# 贪吃蛇游戏 - 规格说明书

## 1. 项目概述

- **项目名称**: 贪吃蛇 (Snake Game)
- **项目类型**: 2D 经典街机游戏 (单 HTML 文件)
- **核心功能**: 玩家控制贪吃蛇在封闭区域内移动，吃食物增长，碰到墙壁或自身则游戏结束
- **目标用户**: 休闲游戏玩家

## 2. 视觉与渲染规格

### 场景设置
- **画布尺寸**: 400x400 像素
- **网格系统**: 20x20 格，每格 20x20 像素
- **背景色**: 深灰色 (#1a1a2e)

### 材质与效果
- **蛇身颜色**: 渐变绿色 (#4ade80 头部 → #22c55e 身体)
- **蛇头**: 带圆角，稍大
- **食物**: 红色圆点 (#ef4444)，带脉冲动画
- **墙壁碰撞线**: 红色警告闪烁效果
- **游戏结束覆盖层**: 半透明黑色 + 居中文字

### 视觉风格
- **整体风格**: 霓虹复古街机风
- **字体**: 等宽像素风格字体 (Press Start 2P 或 monospace fallback)
- **配色**: 深色背景 + 霓虹绿蛇 + 亮红食物

## 3. 游戏机制规格

### 蛇的行为
- **初始长度**: 3 节
- **初始位置**: 画布中央
- **初始方向**: 向右
- **移动速度**: 每 150ms 移动一格
- **增长机制**: 吃到食物后增长 1 节

### 控制方式
- **方向键**: ↑ ↓ ← → 控制移动方向
- **禁止 180° 掉头**: 不能直接反向移动

### 食物机制
- **食物生成**: 随机位置，不能生成在蛇身上
- **分数计算**: 每吃一个食物 +10 分

### 加速道具机制
- **道具外观**: 青色/紫色脉冲圆形，带 » 图标
- **生成概率**: 每次吃食物后 30% 概率生成
- **加速效果**: 移动速度从 150ms 提升至 80ms
- **持续时间**: 5 秒
- **冷却**: 道具被拾取后需重新生成
- **UI 提示**: 左下角显示剩余加速时间

### 碰撞检测
- **墙壁碰撞**: 蛇头碰到画布边界 → 游戏结束
- **自身碰撞**: 蛇头碰到蛇身任意位置 → 游戏结束

## 4. 交互规格

### 用户控制
- 键盘方向键控制蛇移动方向
- 按空格键或回车键重新开始游戏

### UI 元素
- **分数显示**: 左上角实时分数
- **游戏结束提示**: "GAME OVER" + 最终分数 + "按空格键重新开始"

## 5. 验收标准

1. ✅ 蛇能响应上下左右方向键移动
2. ✅ 蛇不能直接掉头（180°反转）
3. ✅ 吃到食物后蛇身增长，分数增加
4. ✅ 碰到墙壁游戏结束，显示 GAME OVER
5. ✅ 碰到自身游戏结束，显示 GAME OVER
6. ✅ 游戏结束后按空格/回车可重新开始
7. ✅ 食物随机生成且不与蛇身重叠
